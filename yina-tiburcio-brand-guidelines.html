06 — Motion

Animación que susurra

Motion al nivel $10K: hand-crafted, intencional. No AOS genérico. Cinco reglas, todas en CSS o Intersection Observer nativo.

Page enter — stagger

Todos los elementos del hero entran con fade + translateY(24px) → 0. Cada hijo con animation-delay de 100ms.

opacity: 0 → 1
transform: translateY(24px) → 0
duration: 700ms · easing: cubic-bezier(.16,1,.3,1)
delay: 0ms, 100ms, 200ms, 300ms...
Card hover — lift

Las property cards se elevan 4px y agregan sombra cálida al hover. Sutil pero perceptible.

transform: translateY(-4px)
box-shadow: 0 12px 32px rgba(42,31,14,.1)
duration: 200ms · easing: ease
Scroll reveal

Secciones entran al viewport con Intersection Observer. Threshold 0.15, una vez.

IntersectionObserver threshold: 0.15
opacity: 0 → 1 · translateY(20px) → 0
duration: 500ms · once: true
Nav scroll shadow

El nav recibe box-shadow cuando scrollY supera los 40px. Comunica profundidad.

window.scrollY > 40
box-shadow: 0 4px 24px rgba(42,31,14,.07)
transition: box-shadow 300ms ease
WhatsApp pulse

El botón flotante late suavemente para llamar atención sin molestar.

@keyframes pulse
0%,100% { transform: scale(1) }
50% { transform: scale(1.05) }
animation: 2s infinite ease-in-out
Lang toggle — swap

El toggle ES/EN hace fade del contenido al cambiar idioma. 200ms, sin layout shift.

opacity: 1 → 0 → 1
duration: 200ms cada fase
easing: ease · no layout shift

ES primero, EN siempre

El español es el idioma principal y el corazón de la marca. El inglés existe para no excluir. Ambos con el mismo tono cálido.

⚠️
Regla bilingüe
Los CTAs siempre en español primero. El toggle ES/EN visible en el nav. Nunca cambiar el tono cálido al inglés — no es una traducción corporativa, es la misma Yina hablando en otro idioma.
